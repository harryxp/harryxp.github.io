Get Number of Displays on Mac

Posted on May 15, 2019

I use chunkwm as my window manager on Mac. It’s pretty awesome but it wreaks havoc on my screen whenever I connect to or disconnect from an external monitor. My silly solution is to write a launchd job that checks the number of displays every 5 seconds and if that changes, restart chunkwm.

One key piece of this solution is to get the number of displays. Here’s the AppleScript I use to do it:

tell application "Image Events"
	launch
	set numDisplays to count displays
	quit
end tell

return numDisplays

Note that it would actually print the number of displays to standard output.
