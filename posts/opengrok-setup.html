OpenGrok Setup

Posted on December 14, 2013

1. Initial setup

$ git clone https://github.com/OpenGrok/OpenGrok.git
$ sudo mkdir -p /var/opengrok/src           # the default location is the easiest
$ sudo mv OpenGrok/ /var/opengrok/src/      # use OpenGrok to index itself

2. Build

Download and install Java 1.7 or higher, and ant.

$ cd /var/opengrok/src/OpenGrok
$ ant clean
$ ant               # default target is 'jar'

3. Deploy web app and start the server

Install tomcat. We need to make the web app running so step 4 can communicate with it.

$ cd /var/opengrok/src/OpenGrok
$ sudo ./OpenGrok deploy

4. Index source files

$ cd /var/opengrok/src/OpenGrok
$ sudo ./OpenGrok index

5. Refresh indexes

$ cd /var/opengrok/src/OpenGrok
$ sudo ./OpenGrok update

APPENDIX A. Development

For serious development, see this page on OpenGrok wiki. Using NetBeans as the core team does makes it much easier to run test cases and debug. If you have trouble installing it, download the zip archive without installer. If you are using a tiling window manager (I use xmonad), you may encounter a Java UI problem. With xmonad you can add startupHook = setWMName "LG3D" to xmonad.hs. Alternatively, start netbeans with _JAVA_AWT_WM_NONREPARENTING=1.
