Tuning Blackberry Passport

Posted on February 19, 2016

I’ve never spent so much time on a phone! But it feels great when you finally got everything perfect.

  • Upgraded to Blackberry OS 10.3.2.2876.
  • Installed Sachesi 2.0.3.
  • Installed Snap using Sachesi.
  • Installed RetroArch 1.3.0 using Snap.
  • RetroArch 1.3.0 ran like a piece of crap.
  • Installed other Android apps using Snap.
  • Some Android apps from Snap didn’t work, requiring “Google Play Services”.
    • Looking for solutions.
    • Followed Cobalt232’s github play store.
    • …and Play Store BB 10 instructions.
    • Installed Google Account Manager 4.3.3, made by Cobalt232.
    • Installed Blackberry Google ID 2.8, made by Cobalt232.
    • Installed Google Play Store 6.0.0 build 1, made by Cobalt232.
    • There’s a problem in the Android runtime on Blackberry OS 10.3.2.2876. Uninstalled apps are not really uninstalled.
      • Followed Cobalt232’s Google Apps Landing Page.
      • Installed Cobalt232’s “Android Settings APK”, saw those apps that failed to uninstall.
      • Followed Android Runtime Analysis.
        • Reverted the Android runtime to an earlier version, using Sachesi.
      • Uninstalled all the Android apps, including RetroArch, using Cobalt232’s app.
    • Installed back patched Android apps, following How to Patch Android Apps.
  • Installed a forked version of RetroArch, from here.
