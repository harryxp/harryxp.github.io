launchd

Posted on May 15, 2019

Apparently cool kids using Mac these days prefer launchd over cron. https://www.launchd.info/ is a good source.

For now, putting this under ~/Library/LaunchAgents is good enough for me.

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
  <key>Label</key>
  <string>harry.job</string>

  <key>ProgramArguments</key>
  <array>
    <string>/path/to/my/script.sh</string>
  </array>

  <key>StartInterval</key>
  <integer>3600</integer>

  <key>RunAtLoad</key>
  <true/>

  <key>StandardErrorPath</key>
  <string>/path/to/stderr.log</string>

  <key>StandardOutPath</key>
  <string>/path/to/stdout.log</string>
</dict>
</plist>

Save it as ~/Library/LaunchAgents/harry.job.plist and run launchctl load ~/Library/LaunchAgents/harry.job.plist to load it.
