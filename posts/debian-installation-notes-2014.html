Debian Installation Notes 2014

Posted on December 13, 2014

TODO try cruft.

  1. Decided to go with Debian jessie (testing) instead of stable since the hardware is really new.

  2. Tried netinstall. Failed. Didn’t recognize wifi card (on Asus Z97 Pro board). Couldn’t install without network.

  3. Downloaded 3 DVDs that cointain jessie. Installed without network. Didn’t install X or any fancy stuff. Just the basics. Went well.

  4. Still no network.

    • lspci -vnn | fgrep Network # see wifi card chipset
    • follow this to install wifi chip driver -
      • http://forum.osxlatitude.com/index.php?/topic/6339-ubuntudebian-14e443b1-broadcom-corporation-device-43b1-rev-03/
      • I was using 6.30.223.248 found at http://www.broadcom.com/support/802.11/linux_sta.php.
      • error from make.log: too few arguments to function cfg80211_ibss_joined
      • patche it using https://513156.bugs.gentoo.org/attachment.cgi?id=378836

  5. Installed wicd to log into wifi. Hooray!

  6. Used netselect-apt to choose a fast debian mirror. Optional.

  7. Installed X and xmonad. Basically following debianelectronics.blogspot.com/2012/08/xmonad-on-fresh-debian-install-part-1.html

  8. GTX 970 video card driver.

    • Drivers from repositories didn’t work, even non-free ones.
    • Card is too new. Used NVIDIA-Linux-x86_64-343.22.run executable from NVIDIA website.
    • Had gcc 4.9 rather than 4.8. Had to do this to build:
    • sudo env IGNORE_CC_MISMATCH=1 ./NVIDIA-Linux-x86_64-343.22.run
    • Multiple displays: nvidia-settings

  9. Installed Google Chrome from Google’s repository.

  10. Installed Dropbox from https://www.dropbox.com/install?os=lnx. There is a repository but that version has a bug and won’t start.

    $ ~/.dropbox-dist/dropboxd

  11. To get a good monospaced font for xterm, I followed the instruction at https://gist.github.com/rogerleite/99819

    $ cd /usr/share/fonts/truetype/
$ sudo mkdir ttf-monaco
$ cd ttf-monaco/
$ sudo wget http://www.gringod.com/wp-upload/software/Fonts/Monaco_Linux.ttf
$ sudo mkfontdir
$ cd ..
$ fc-cache
$ # here you need to edit ~/Xresources first to use Monaco
$ xrdb -merge ~/.Xresources
